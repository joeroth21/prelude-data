prelude-data — static feed

Versioned JSON powering PRELUDE. Publicly available factual data compiled for educational display. No recommendations, no ratings, no trade signals.

  • feed/index.json — version index
  • feed/v1/feed_meta.json — freshness manifest
  • feed/v1/companies.json — curated company universe
  • feed/v1/pipeline.json — S-1 pipeline (SEC EDGAR)
  • feed/v1/wrappers.json — retail-accessible wrappers
  • feed/v1/signals.json — availability + news links

Every datum carries source_url and as_of. See the repository README for sources, terms-of-use posture, and refresh cadence.
